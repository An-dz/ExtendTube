<static> control

Issue a command to video player.
Parameters:
Name Type Argument Description
command String Tell player what to do. This argument is enumerated and can be: play/pause, pause, play, stop, seek or volume. Command play/pause will play or pause video depending on current state. Command stop will pause video and stop buffering it. Commands seekvolume will seek or change volume for value given by second argument.
data String | Number <optional>
 This argument is needed when issuing commands seek and volume. If this argument is number or string written as <number>% (ex. 73%) change will be absolute regardless of current state. If argument is string written as +<number>% or -<number>% (ex. +38% or -49%) change will be relative to current state.
