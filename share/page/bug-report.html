ExtendTube: Report a bug!

If you encounter a bug in ExtendTube, or ExtendTube is causing some abnormal behaviour of YouTube, you should report it. In your bug report you should include:

  • bug description and how to reproduce it,
  • URI of page where bug is occurred,
  • debug log for ExtendTube (click here and then "Export logs" button),
  • TestTube features you are subscribed to (like HTML5),
  • list of extensions you have installed that also run on YouTube pages,
  • list of user JavaScripts you have installed that run on YouTube pages.

Optionally you can include:

  • current preferences for ExtendTube (click here and then "Export" button),
  • Opera version and platform,
  • any other information you think might be relevant.

There are four places where you can report a bug:

  • support page
  • ExtendTube project page on GitHub
  • via e-mail
  • official extension's page

Report via e-mail is preferred as you can attach file(s) mentioned above.
